【游戏简介】
这是一款支持单人和双人模式的贪吃蛇游戏，具有炫酷的视觉效果和音效。

【系统要求】
- Python 3.6 或更高版本
- 现代浏览器（Chrome、Firefox、Safari等）

【安装说明】
1. 确保已安装Python 3.6+
2. 安装太麻烦，直接使用本人新开的网页
   https://snake-game-xi-swart.vercel.app/

【游戏控制】
玩家1：方向键 (↑ ↓ ← →)
玩家2：WASD 键 (W A S D)
空格键：暂停/继续游戏

【移动设备】
支持触摸控制，自动显示虚拟摇杆

祝您游戏愉快！
