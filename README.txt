炫彩贪吃蛇游戏 - 使用说明

【游戏简介】
这是一款支持单人和双人模式的贪吃蛇游戏，具有炫酷的视觉效果和音效。

【系统要求】
- Python 3.6 或更高版本
- 现代浏览器（Chrome、Firefox、Safari等）

【安装说明】
1. 确保已安装Python 3.6+
2. 双击运行 run.bat (Windows) 或 run.sh (Mac/Linux)

【游戏控制】
玩家1：方向键 (↑ ↓ ← →)
玩家2：WASD 键 (W A S D)
空格键：暂停/继续游戏

【移动设备】
支持触摸控制，自动显示虚拟摇杆

【注意事项】
- 首次运行会自动安装所需依赖
- 游戏运行时请勿关闭命令窗口
- 访问 http://localhost:5000 开始游戏

【退出游戏】
在命令窗口中按 Ctrl+C

祝您游戏愉快！
